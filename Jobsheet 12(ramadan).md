Jawaban Petanyaan 12.2.3
1. * Single Linked List
Setiap node hanya memiliki satu pointer yaitu next.
Traversal hanya dapat dilakukan dari depan ke belakang.
Struktur lebih sederhana dan penggunaan memori lebih sedikit.

* Double Linked List
Setiap node memiliki dua pointer yaitu next dan prev.
Traversal dapat dilakukan dua arah:
 - dari head ke tail
 - dari tail ke head
Membutuhkan memori lebih besar karena memiliki dua pointer.

2. * Fungsi **next**:
- membantu traversal maju
- membantu proses penambahan node di belakang
- membantu penghapusan node

* Fungsi **prev**:
- membantu traversal mundur
- mempermudah penghapusan node
- mempermudah penyisipan node di tengah

3. Fungsinya:
- menginisialisasi linked list dalam keadaan kosong
- head = null berarti belum ada node pertama
- tail = null berarti belum ada node terakhir

4. Karena saat linked list kosong dan ditambahkan satu node:
- node tersebut menjadi node pertama
- sekaligus menjadi node terakhir

maka:
- head menunjuk node pertama
- tail menunjuk node terakhir
